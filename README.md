## What is ArtistHub?

ArtistHub is a full-featured social platform built for artists вҖ” a place to post work, grow a following, chat with other creatives in real-time, and accept donations from fans. It combines the portfolio features of Behance, the social layer of Instagram, and the community feel of Discord into a single self-hostable app.

Upload a painting, get AI-generated style tags, collect likes and comments, and chat with your followers вҖ” all without leaving the platform.

## Feature Overview

<br />

### Gallery & Portfolio

Artists get a full profile page with their work in a masonry grid. Every piece supports a title, description, category, and tags вҖ” with an AI tagging system (CLIP vision model + ColorThief) that analyzes images on upload and suggests art style labels and dominant color names automatically. Collections let artists curate named portfolios grouped into a grid on their profile.

<br />

### Explore & Discovery

| | Feature | Detail |
|---|---|---|
| **вҹЎ** | Trending row | Top 6 most-liked artworks surfaced at the top of Explore |
| **вҹЎ** | Category filter | Digital Art В· Painting В· Photography В· Sculpture В· Mixed Media В· Street Art В· and more |
| **вҹЎ** | Artist directory | Browse all artists sorted by newest or most followed |
| **вҹЎ** | Full-text search | Searches artwork titles, descriptions, and artist bios simultaneously |
| **вҹЎ** | Infinite scroll | Feed, Explore, and profiles all paginate via AJAX вҖ” no page reloads |

<br />

### Social Layer

Follow artists, like artwork, leave comments. A notification bell tracks all activity вҖ” new followers, likes, comments, and direct messages вҖ” with a live unread count polling every 30 seconds. Verified artist badges are admin-assignable.

<br />

### Community Channels

Public chat channels with a feel close to Discord. Messages update live every 5 seconds. Images can be attached to any message. Moderators get a full toolkit:

```
pin messages      вҶ’ stays pinned at the top of the channel for all users
delete messages   вҶ’ removed instantly via an in-memory sync registry  
kick users        вҶ’ session-level removal with an optional reason
ban users         вҶ’ permanent channel ban, manageable from admin panel
@mention          вҶ’ autocomplete pulls from recent channel participants
report messages   вҶ’ notifies all admins immediately
```

<br />

### Direct Messages

Private one-to-one conversations between any two users. Enter to send, Shift+Enter for a new line. Read receipts mark messages as seen when a conversation is opened. Unread counts appear in the nav.

<br />

### Donations via Stripe

Every artist gets a personal donation page at `/donate/username`. Fans can make one-time payments or set up recurring subscriptions. Donors receive a profile badge. Stripe payment intents, subscriptions, and webhooks are fully wired.

<br />

### Admin Dashboard

```
/admin/users      вҶ’  search В· ban/unban В· change roles В· delete accounts
/admin/channels   вҶ’  create channels В· view history В· manage bans
/admin/reports    вҶ’  review flagged content В· resolve or dismiss
/admin/analytics  вҶ’  growth charts В· top artwork В· art medium breakdown
/admin/error-logs вҶ’  server error viewer with clear/delete
```

## Getting Started

### Prerequisites

- Python 3.11+
- A PostgreSQL database вҖ” [Supabase](https://supabase.com) free tier works perfectly
- A [Cloudinary](https://cloudinary.com) account for image hosting
- A [Stripe](https://stripe.com) account if you want donations enabled

<br />

### 1 вҖ” Clone and install

```bash
git clone https://github.com/your-username/artistshub.git
cd artistshub
pip install -r requirements.txt
```

<br />

### 2 вҖ” Configure environment

Create a `.env` file in the project root:

```env
# Required
DATABASE_URL=postgresql://user:password@host:5432/dbname
SECRET_KEY=a-long-random-secret-key

# Required for image uploads
CLOUDINARY_CLOUD_NAME=your-cloud-name
CLOUDINARY_API_KEY=your-api-key
CLOUDINARY_API_SECRET=your-api-secret

# Optional вҖ” only needed for donation pages
STRIPE_PUBLIC_KEY=pk_live_...
STRIPE_SECRET_KEY=sk_live_...
STRIPE_WEBHOOK_SECRET=whsec_...
```

<br />

### 3 вҖ” Set up the database

**Using Supabase?** Run the migration file in your Supabase SQL editor:

```
supabase/migrations/20260409185033_create_all_app_tables.sql
```

**Using your own Postgres?** Flask-Migrate handles it:

```bash
flask db upgrade
```

<br />

### 4 вҖ” Run

```bash
python run.py
```

Visit `http://localhost:5000`. Register your account, then promote yourself to admin directly in the database:

```sql
UPDATE users SET is_admin = true WHERE username = 'your-username';
```

From there you can create channels and manage everything from `/admin`.

## AI Auto-Tagging

When artwork is uploaded, ArtistHub runs two models against the image before the form is submitted:

**CLIP ViT-B/32** classifies the image against 20 art style labels with a 0.2 confidence threshold вҖ” detecting styles like impressionism, concept art, pixel art, watercolor, and street art.

**ColorThief** extracts up to 5 dominant colors from the image and maps RGB values to human-readable color names like *dusty rose* or *slate blue*.

Tag suggestions arrive via a `POST /artwork/preview-tags` AJAX call вҖ” the upload is never blocked if the AI step fails. Artists can accept, remove, or add their own tags up to a maximum of 10. The CLIP model is cached in memory after first load, so subsequent uploads are fast.
